# Drone Detection and Identification Using QR Codes

## Overview
This project is a computer vision-based drone identification system that uses QR-code markers attached to drones for real-time detection and recognition. The system captures video frames, detects QR codes, decodes the embedded information, and displays the drone identity with visual annotations.

## Features
- Real-time video processing
- QR-code detection and decoding
- Drone identification and labeling
- Bounding box visualization
- Lightweight and cost-effective implementation
- OpenCV-based computer vision pipeline

## Technologies Used
- Python
- OpenCV
- NumPy

## Working
1. Capture video frames using a camera.
2. Detect QR codes present in the frame.
3. Decode the information stored in the QR code.
4. Identify the drone based on the decoded data.
5. Display the drone information along with a bounding box.

## Applications
- Drone fleet management
- Indoor drone tracking
- Educational UAV systems
- Controlled airspace monitoring
- Smart surveillance systems
